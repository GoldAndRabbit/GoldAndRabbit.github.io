иҝ”еӣһдҪ еҸҜд»Ҙд»Һиҝҷз¬”дәӨжҳ“дёӯиҺ·еҸ–зҡ„жңҖеӨ§еҲ©ж¶ҰгҖӮеҰӮжһңдҪ дёҚиғҪиҺ·еҸ–д»»дҪ•еҲ©ж¶ҰпјҢиҝ”еӣһ 0 гҖӮ

зӨәдҫӢ 1пјҡиҫ“е…Ҙпјҡ[7,1,5,3,6,4] иҫ“еҮәпјҡ5
и§ЈйҮҠпјҡеңЁз¬¬ 2 еӨ©пјҲиӮЎзҘЁд»·ж ј = 1пјүзҡ„ж—¶еҖҷд№°е…ҘпјҢеңЁз¬¬ 5 еӨ©пјҲиӮЎзҘЁд»·ж ј = 6пјүзҡ„ж—¶еҖҷеҚ–еҮәпјҢжңҖеӨ§еҲ©ж¶Ұ = 6-1 = 5 гҖӮ
жіЁж„ҸеҲ©ж¶ҰдёҚиғҪжҳҜ 7-1 = 6, еӣ дёәеҚ–еҮәд»·ж јйңҖиҰҒеӨ§дәҺд№°е…Ҙд»·ж јпјӣеҗҢж—¶пјҢдҪ дёҚиғҪеңЁд№°е…ҘеүҚеҚ–еҮәиӮЎзҘЁгҖӮ

зӨәдҫӢ 2пјҡиҫ“е…Ҙпјҡprices = [7,6,4,3,1] иҫ“еҮәпјҡ0 и§ЈйҮҠпјҡеңЁиҝҷз§Қжғ…еҶөдёӢ, жІЎжңүдәӨжҳ“е®ҢжҲҗ, жүҖд»ҘжңҖеӨ§еҲ©ж¶Ұдёә 0гҖӮ

еҲҶжһҗ:
1.еӣ дёәеҸӘйҖү 1 ж¬Ўд№°е…Ҙж—¶й—ҙе’Ң 1 ж¬Ў еҚ–еҮәж—¶й—ҙ, дё”еҚ–еҮәж—¶й—ҙеҝ…з„¶еңЁд№°е…Ҙд№ӢеҗҺиҮіе°‘дёҖеӨ©, жүҖд»ҘжҲ‘д»¬йҖҡиҝҮйҒҚеҺҶеҚ–еҮәж—¶й—ҙжқҘжүҫжңҖеӨ§еҲ©ж¶Ұ
2.жҲ‘д»¬йҒҚеҺҶзҡ„жҳҜз¬¬ i еӨ©дҪңдёәеҚ–еҮәзӮ№зҡ„зҡ„жңҖеӨ§еҲ©ж¶Ұ, еӣ дёәд№°е…ҘзӮ№ж°ёиҝңеңЁеҚ–еҮәзӮ№д№ӢеүҚ, йҒҚеҺҶеҗҢжӯҘз»ҙжҠӨеҺҶеҸІжңҖдҪҺд№°е…ҘзӮ№ 

class Solution {
 public:
  int maxProfit(vector<int>& prices) {
    int len = prices.size();
    if (len == 0) {
      return 0;
    }
    int ans = INT_MIN;
    // з»ҙжҠӨз¬¬ i еӨ©д№ӢеүҚзҡ„еҺҶеҸІжңҖдҪҺзӮ№
    int preMin = prices[0];
    for (int i = 1; i < prices.size(); ++i) {
      ans = max(ans, prices[i] - preMin);
      preMin = min(preMin, prices[i]);
    }
    return (ans < 0) ? 0 : ans;
  }
};

4.иҫ№з•ҢжқЎд»¶, йңҖиҰҒзЎ®е®ҡеҲқе§Ӣзҡ„жқЎд»¶, д№ҹе°ұжҳҜз¬¬ 0 еӨ© (зңҹе®һж„Ҹд№үдёӢзҡ„з¬¬ 1 еӨ©) зҡ„жңҖеӨ§еҲ©ж¶ҰжҳҜд»Җд№Ҳ?
dp[0][0] = 0, з¬¬ 0 еӨ©, жІЎжңүжҢҒжңүзҡ„ж—¶еҖҷ, еҲ©ж¶Ұдёә 0
dp[0][1] = -prices[0], з¬¬ 0 еӨ©з»“жқҹд№°дәҶиӮЎзҘЁзҡ„ж—¶еҖҷ, еҲ©ж¶Ұдёә-prices[0] 

class Solution {
 public:
  int maxProfit(vector<int>& prices) {
    int n = prices.size();
    vector<vector<int>> dp(n, vector<int>(2));
    dp[0][0] = 0;
    dp[0][1] = -prices[0];
    for (int i = 1; i < n; ++i){
      dp[i][0] = max(dp[i-1][0], dp[i-1][1] + prices[i]);
      dp[i][1] = max(dp[i-1][1], dp[i-1][0] - prices[i]);
    }
    return dp[n-1][0];
  }
};

class Solution {

    vector<int> hold(n);

      nohold[i] = max(nohold[i-1], hold[i-1] + prices[i]);
    }
    return nohold[n-1];
  }
};

nohold[0][2] з¬¬ 0 еӨ©жІЎжҢҒиӮЎ, дәӨжҳ“ж¬Ўж•°2, дёҚеҗҲжі•зҠ¶жҖҒ, и®ҫзҪ®дёҖдёӘжңҖе°ҸеҖј INT_MIN/2
nohold[0][0] з¬¬ 0 еӨ©жІЎжҢҒиӮЎ, дәӨжҳ“ж¬Ўж•°0, еҗҲжі•зҡ„зҠ¶жҖҒ, еҲ©ж¶ҰжҳҜ0

class Solution {
 public:
  int maxProfit(int k, vector<int>& prices) {
    int n = prices.size();
    vector<vector<int>> hold(n, vector<int>(k+1));
    vector<vector<int>> nohold(n, vector<int>(k+1));
    // з¬¬ 0 е·Із»ҸжҢҒиӮЎдёӢзҡ„ 0 з¬”дәӨжҳ“, еҸӘжңүдёҖз§ҚеҸҜиғҪжҳҜд№°дәҶ prices[0], жңҖеӨ§еҲ©ж¶Ұ -prices[0]
    hold[0][0] = -prices[0];
    // з¬¬ 1 жңӘжҢҒиӮЎдёӢзҡ„ 0 з¬”дәӨжҳ“, еҸӘжңүдёҖз§ҚеҗҲжі•еҸҜиғҪжҳҜд»Җд№ҲйғҪжІЎеҒҡ, жңҖеӨ§еҲ©ж¶Ұ 0 
    nohold[0][0] = 0;
    // ж Үи®°жүҖжңүдёҚеҗҲжі•зҡ„зҠ¶жҖҒдёәдёҖдёӘиҫғе°Ҹзҡ„еҖј, жҜ”еҰӮ INT_MIN / 2
    for (int i = 1; i <= k; ++i) {
      hold[0][i]   = INT_MIN / 2;
      nohold[0][i] = INT_MIN / 2;
    }
    // жһҡдёҫеӨ©ж•°
    for (int i = 1; i < n; ++i) {
      hold[i][0] = max(hold[i-1][0], nohold[i-1][0] - prices[i]); 
      for (int j = 1; j <= k; ++j) {
        hold[i][j] = max(hold[i-1][j], nohold[i-1][j] - prices[i]); 
        nohold[i][j] = max(nohold[i-1][j], hold[i-1][j-1] + prices[i]);
      }
    }
    return *max_element(nohold[n-1].begin(), nohold[n-1].end());
  }
};

з»ҷе®ҡдёҖдёӘж•°з»„пјҢе®ғзҡ„з¬¬ i дёӘе…ғзҙ жҳҜдёҖж”Ҝз»ҷе®ҡзҡ„иӮЎзҘЁеңЁз¬¬ i еӨ©зҡ„д»·ж јгҖӮи®ҫи®ЎдёҖдёӘз®—жі•жқҘи®Ўз®—дҪ жүҖиғҪиҺ·еҸ–зҡ„жңҖеӨ§еҲ©ж¶ҰгҖӮдҪ жңҖеӨҡеҸҜд»Ҙе®ҢжҲҗ дёӨз¬” дәӨжҳ“гҖӮ
жіЁж„ҸпјҡдҪ дёҚиғҪеҗҢж—¶еҸӮдёҺеӨҡз¬”дәӨжҳ“пјҲдҪ еҝ…йЎ»еңЁеҶҚж¬Ўиҙӯд№°еүҚеҮәе”®жҺүд№ӢеүҚзҡ„иӮЎзҘЁпјүгҖӮ

зӨәдҫӢ 1: иҫ“е…Ҙпјҡprices = [3,3,5,0,0,3,1,4] иҫ“еҮәпјҡ6
и§ЈйҮҠпјҡеңЁз¬¬ 4 еӨ©пјҲиӮЎзҘЁд»·ж ј = 0пјүзҡ„ж—¶еҖҷд№°е…ҘпјҢеңЁз¬¬ 6 еӨ©пјҲиӮЎзҘЁд»·ж ј = 3пјүзҡ„ж—¶еҖҷеҚ–еҮәпјҢиҝҷз¬”дәӨжҳ“жүҖиғҪиҺ·еҫ—еҲ©ж¶Ұ = 3-0 = 3 гҖӮ
йҡҸеҗҺпјҢеңЁз¬¬ 7 еӨ©пјҲиӮЎзҘЁд»·ж ј = 1пјүзҡ„ж—¶еҖҷд№°е…ҘпјҢеңЁз¬¬ 8 еӨ© пјҲиӮЎзҘЁд»·ж ј = 4пјүзҡ„ж—¶еҖҷеҚ–еҮәпјҢиҝҷз¬”дәӨжҳ“жүҖиғҪиҺ·еҫ—еҲ©ж¶Ұ = 4-1 = 3 гҖӮ

зӨәдҫӢ 2пјҡиҫ“е…Ҙпјҡprices = [1,2,3,4,5] иҫ“еҮәпјҡ4
и§ЈйҮҠпјҡеңЁз¬¬ 1 еӨ©пјҲиӮЎзҘЁд»·ж ј = 1пјүзҡ„ж—¶еҖҷд№°е…ҘпјҢеңЁз¬¬ 5 еӨ© пјҲиӮЎзҘЁд»·ж ј = 5пјүзҡ„ж—¶еҖҷеҚ–еҮә, иҝҷз¬”дәӨжҳ“жүҖиғҪиҺ·еҫ—еҲ©ж¶Ұ = 5-1 = 4 гҖӮ
жіЁж„ҸдҪ дёҚиғҪеңЁз¬¬ 1 еӨ©е’Ңз¬¬ 2 еӨ©жҺҘиҝһиҙӯд№°иӮЎзҘЁпјҢд№ӢеҗҺеҶҚе°Ҷе®ғд»¬еҚ–еҮәгҖӮ
еӣ дёәиҝҷж ·еұһдәҺеҗҢж—¶еҸӮдёҺдәҶеӨҡз¬”дәӨжҳ“пјҢдҪ еҝ…йЎ»еңЁеҶҚж¬Ўиҙӯд№°еүҚеҮәе”®жҺүд№ӢеүҚзҡ„иӮЎзҘЁгҖӮ

зӨәдҫӢ 3пјҡиҫ“е…Ҙпјҡprices = [7,6,4,3,1] иҫ“еҮәпјҡ0 и§ЈйҮҠпјҡеңЁиҝҷдёӘжғ…еҶөдёӢ, жІЎжңүдәӨжҳ“е®ҢжҲҗ, жүҖд»ҘжңҖеӨ§еҲ©ж¶Ұдёә 0гҖӮ

зӨәдҫӢ 4пјҡиҫ“е…Ҙпјҡprices = [1] иҫ“еҮәпјҡ0

еҲҶжһҗ:
1.жӣҙдёҖиҲ¬зҡ„еҲҶжһҗиҜҰжғ…и§Ғ 188.д№°еҚ–иӮЎзҘЁзҡ„жңҖдҪіж—¶жңә IV 

class Solution {
 public:
  int maxProfit(vector<int>& prices) {
    int n = prices.size();
    int k = 2;
    vector<vector<int>> hold(n, vector<int>(k+1));
    vector<vector<int>> nohold(n, vector<int>(k+1));
    // з¬¬ 0 е·Із»ҸжҢҒиӮЎдёӢзҡ„ 0 з¬”дәӨжҳ“, еҸӘжңүдёҖз§ҚеҸҜиғҪжҳҜд№°дәҶ prices[0], жңҖеӨ§еҲ©ж¶Ұ -prices[0]
    hold[0][0] = -prices[0];
    // з¬¬ 1 жңӘжҢҒиӮЎдёӢзҡ„ 0 з¬”дәӨжҳ“, еҸӘжңүдёҖз§ҚеҗҲжі•еҸҜиғҪжҳҜд»Җд№ҲйғҪжІЎеҒҡ, жңҖеӨ§еҲ©ж¶Ұ 0 
    nohold[0][0] = 0;
    // ж Үи®°жүҖжңүдёҚеҗҲжі•зҡ„зҠ¶жҖҒдёәдёҖдёӘиҫғе°Ҹзҡ„еҖј, жҜ”еҰӮ INT_MIN / 2
    for (int i = 1; i <= k; ++i) {
      hold[0][i]   = INT_MIN / 2;
      nohold[0][i] = INT_MIN / 2;
    }
    // жһҡдёҫеӨ©ж•°
    for (int i = 1; i < n; ++i) {
      hold[i][0] = max(hold[i-1][0], nohold[i-1][0] - prices[i]); 
      for (int j = 1; j <= k; ++j) {
        hold[i][j] = max(hold[i-1][j], nohold[i-1][j] - prices[i]); 
        nohold[i][j] = max(nohold[i-1][j], hold[i-1][j-1] + prices[i]);
      }
    }
    return *max_element(nohold[n-1].begin(), nohold[n-1].end());
  }
};

309.жңҖдҪід№°еҚ–иӮЎзҘЁж—¶жңәеҗ«еҶ·еҶ»жңҹ

з»ҷе®ҡдёҖдёӘж•ҙж•°ж•°з»„pricesпјҢе…¶дёӯз¬¬ prices[i] иЎЁзӨәз¬¬ i еӨ©зҡ„иӮЎзҘЁд»·ж ј гҖӮвҖӢ
и®ҫи®ЎдёҖдёӘз®—жі•и®Ўз®—еҮәжңҖеӨ§еҲ©ж¶ҰгҖӮеңЁж»Ўи¶ід»ҘдёӢзәҰжқҹжқЎд»¶дёӢпјҢдҪ еҸҜд»Ҙе°ҪеҸҜиғҪең°е®ҢжҲҗжӣҙеӨҡзҡ„дәӨжҳ“пјҲеӨҡж¬Ўд№°еҚ–дёҖж”ҜиӮЎзҘЁпјү:
еҚ–еҮәиӮЎзҘЁеҗҺпјҢдҪ ж— жі•еңЁз¬¬дәҢеӨ©д№°е…ҘиӮЎзҘЁ (еҚіеҶ·еҶ»жңҹдёә 1 еӨ©)гҖӮ
жіЁж„ҸпјҡдҪ дёҚиғҪеҗҢж—¶еҸӮдёҺеӨҡз¬”дәӨжҳ“пјҲдҪ еҝ…йЎ»еңЁеҶҚж¬Ўиҙӯд№°еүҚеҮәе”®жҺүд№ӢеүҚзҡ„иӮЎзҘЁпјүгҖӮ

зӨәдҫӢ 1: иҫ“е…Ҙ: prices = [1,2,3,0,2] иҫ“еҮә: 3
и§ЈйҮҠ: еҜ№еә”зҡ„дәӨжҳ“зҠ¶жҖҒдёә: [д№°е…Ҙ, еҚ–еҮә, еҶ·еҶ»жңҹ, д№°е…Ҙ, еҚ–еҮә]

зӨәдҫӢ 2: иҫ“е…Ҙ: prices = [1] иҫ“еҮә: 0

еҲҶжһҗ:
1.зҠ¶жҖҒеҲҶжһҗ, зӣёжҜ”жІЎжңүеҶ·еҶ»жңҹзҡ„жғ…еҶө, еҶ·еҶ»жңҹзҡ„еҮәзҺ°еҸӘжңүеҮәзҺ°жҢҒжңүиӮЎзҘЁзҡ„жғ…еҶө
2.жҲ‘д»¬з”Ё dp[i][j] жқҘзј–з Ғ з¬¬ i еӨ©жҢҒжңүжҲ–иҖ…жңӘжҢҒжңүдё”е…јйЎҫеҗҺдёҖеӨ©жҳҜеҗҰеҶ·еҶ»зҡ„жқЎд»¶дёӢзҡ„жңҖеӨ§еҲ©ж¶Ұжғ…еҶө, е…·дҪ“жқҘиҜҙ:
dp[i][0] иЎЁзӨә i еӨ©з»“жқҹжҢҒжңүиӮЎзҘЁ
dp[i][1] иЎЁзӨә i еӨ©з»“жқҹжІЎжҢҒжңүиӮЎзҘЁ, дё”д№ӢеҗҺдёҖеӨ©жҳҜеҶ·еҶ»жңҹ
dp[i][2] иЎЁзӨә i еӨ©з»“жқҹжІЎжҢҒжңүиӮЎзҘЁ, дё”д№ӢеҗҺдёҖеӨ©дёҚжҳҜеҶ·еҶ»жңҹ
3.жҲ‘д»¬еҲҶзұ»и®Ёи®әдёҖдёӢжүҖжңүеҸҜиғҪзҡ„иҪ¬з§», йңҖиҰҒдёҖзӮ№дёҖзӮ№жқҘзңӢ 

// еҪ“еӨ©з»“жқҹжҢҒжңүиӮЎзҘЁзҡ„зҠ¶жҖҒ
dp[i][0] = max(
  dp[i-1][0],             // жҳЁеӨ©жҢҒжңүиӮЎзҘЁ, д»ҠеӨ©д»Җд№Ҳд№ҹжІЎе№Ізҡ„зҠ¶жҖҒ, д»ҺжҳЁеӨ©жҢҒжңүиӮЎзҘЁиҝҮжқҘ
  dp[i-1][2] - prices[i]; // жҳЁеӨ©жІЎжҢҒжңүиӮЎзҘЁ, д»ҠеӨ©д№°дәҶзҡ„зҠ¶жҖҒ, д»ҺжҳЁеӨ©жІЎжңүжҢҒжңүз»“жқҹдё”еҗҺдёҖеӨ©дёҚжҳҜеҶ·еҶ»жңҹ иҪ¬з§»иҝҮжқҘ
)
// еҪ“еӨ©з»“жқҹжІЎжңүжҢҒжңүиӮЎзҘЁ, дё”д№ӢеҗҺеңЁеҶ·еҶ»жңҹзҡ„, е°ұдёҖз§Қжғ…еҶө
dp[i][1] = max(
  dp[i-1][0] + prides[i]  // жҳЁеӨ©жҢҒжңүиӮЎзҘЁ, д»ҠеӨ©еҚ–дәҶ, д№ӢеҗҺеҶ·еҶ»жңҹ
)
// еҪ“еӨ©з»“жқҹжІЎжңүжҢҒжңүиӮЎзҘЁ, дё”д№ӢеҗҺдёҚеңЁеҶ·еҶ»жңҹ
dp[i][2] = max(
  dp[i-1][1]            // жҳЁеӨ©жІЎжңүжҢҒжңүиӮЎзҘЁ, д»ҠеӨ©жҳҜеҶ·еҶ»жңҹ
  dp[i-1][2]            // жҳЁеӨ©жІЎжңүжҢҒжңүиӮЎзҘЁ, д»ҠеӨ©д№ҹдёҚжҳҜеҶ·еҶ»жңҹ
)

4.иҫ№з•ҢжқЎд»¶
dp[0][0] з¬¬ 0 еӨ©жҢҒжңүиӮЎзҘЁ, еҗҲжі•жғ…еҶөжҳҜз¬¬ 0 еӨ©д№°дәҶ prices[0], еҲ©ж¶Ұ -prices[0]
dp[0][1] з¬¬ 0 еӨ©жңӘжҢҒжңүиӮЎзҘЁ, дё”еҗҺдёҖеӨ©жҳҜеҶ·еҶ»жңҹ, дёҚеҗҲжі•, еҲ©ж¶Ұ INT_MIN / 2;
dp[0][2] з¬¬ 0 еӨ©жңӘжҢҒжңүиӮЎзҘЁ, дё”еҗҺдёҖеӨ©дёҚжҳҜеҶ·еҶ»жңҹ, еҲ©ж¶Ұ 0

class Solution {
 public:
  int maxProfit(vector<int>& prices) {
    int len = prices.size();
    vector<vector<int>> dp(len, vector<int>(3));
    // dp[0][0] з¬¬ 0 еӨ©жҢҒжңүиӮЎзҘЁ, еҗҲжі•жғ…еҶөжҳҜз¬¬ 0 еӨ©д№°дәҶ prices[0], еҲ©ж¶Ұ -prices[0]
    // dp[0][1] з¬¬ 0 еӨ©жңӘжҢҒжңүиӮЎзҘЁ, дё”еҗҺдёҖеӨ©жҳҜеҶ·еҶ»жңҹ, дёҚеҗҲжі•, еҲ©ж¶Ұ INT_MIN / 2;
    // dp[0][2] з¬¬ 0 еӨ©жңӘжҢҒжңүиӮЎзҘЁ, дё”еҗҺдёҖеӨ©дёҚжҳҜеҶ·еҶ»жңҹ, еҲ©ж¶Ұ 0
    dp[0][0] = -prices[0];
    dp[0][1] = INT_MIN / 2;
    dp[0][2] = 0;
    for (int i = 1; i < len; ++i) {
      dp[i][0] = max(dp[i-1][0], dp[i-1][2] - prices[i]);
      dp[i][1] = dp[i-1][0] + prices[i];
      dp[i][2] = max(dp[i-1][1], dp[i-1][2]);
    }
    return max(dp[len-1][1], dp[len-1][2]);
  }
};

714.д№°еҚ–иӮЎзҘЁзҡ„жңҖдҪіж—¶жңәеҗ«жүӢз»ӯиҙ№

з»ҷе®ҡдёҖдёӘж•ҙж•°ж•°з»„ pricesпјҢе…¶дёӯ prices[i] иЎЁзӨәз¬¬ i еӨ©зҡ„иӮЎзҘЁд»·ж ј пјӣж•ҙж•° fee д»ЈиЎЁдәҶдәӨжҳ“иӮЎзҘЁзҡ„жүӢз»ӯиҙ№з”ЁгҖӮ
дҪ еҸҜд»Ҙж— йҷҗж¬Ўең°е®ҢжҲҗдәӨжҳ“пјҢдҪҶжҳҜдҪ жҜҸз¬”дәӨжҳ“йғҪйңҖиҰҒд»ҳжүӢз»ӯиҙ№гҖӮеҰӮжһңдҪ е·Із»Ҹиҙӯд№°дәҶдёҖдёӘиӮЎзҘЁпјҢеңЁеҚ–еҮәе®ғд№ӢеүҚдҪ е°ұдёҚиғҪеҶҚз»§з»ӯиҙӯд№°иӮЎзҘЁдәҶгҖӮ
иҝ”еӣһиҺ·еҫ—еҲ©ж¶Ұзҡ„жңҖеӨ§еҖјгҖӮжіЁж„ҸпјҡиҝҷйҮҢзҡ„дёҖз¬”дәӨжҳ“жҢҮд№°е…ҘжҢҒжңүе№¶еҚ–еҮәиӮЎзҘЁзҡ„ж•ҙдёӘиҝҮзЁӢпјҢжҜҸз¬”дәӨжҳ“дҪ еҸӘйңҖиҰҒдёәж”Ҝд»ҳдёҖж¬ЎжүӢз»ӯиҙ№гҖӮ

зӨәдҫӢ 1пјҡиҫ“е…Ҙпјҡprices = [1, 3, 2, 8, 4, 9], fee = 2 иҫ“еҮәпјҡ8 и§ЈйҮҠпјҡиғҪеӨҹиҫҫеҲ°зҡ„жңҖеӨ§еҲ©ж¶Ұ:
еңЁжӯӨеӨ„д№°е…Ҙ prices[0] = 1
еңЁжӯӨеӨ„еҚ–еҮә prices[3] = 8
еңЁжӯӨеӨ„д№°е…Ҙ prices[4] = 4
еңЁжӯӨеӨ„еҚ–еҮә prices[5] = 9
жҖ»еҲ©ж¶Ұ: ((8 - 1) - 2) + ((9 - 4) - 2) = 8

зӨәдҫӢ 2пјҡиҫ“е…Ҙпјҡprices = [1,3,7,5,10,3], fee = 3 иҫ“еҮәпјҡ6

еҲҶжһҗ:
1.е°ҶжүӢз»ӯиҙ№з®—еңЁжңҖеӨ§еҲ©ж¶Ұзҡ„зҠ¶жҖҒиҪ¬з§»йҮҢйқў, еҪ“дё”д»…еҪ“еҚ–еҮәзҡ„ж—¶еҖҷжүҚз®—жүӢз»ӯиҙ№
nohold[i] = max(nohold[i-1], hold[i-1] + prices[i] - fee)
hold[i] = max(hold[i-1], nohold[i-1] - pricer[i])
2.иҫ№з•ҢжқЎд»¶
nohold[0] = 0 з¬¬ 0 еӨ©жІЎд№°е…Ҙ
hold[0] = -prices[0] з¬¬ 0 еӨ©д№°е…Ҙ 

class Solution {
 public:
  int maxProfit(vector<int>& prices, int fee) {
    int len = prices.size();
    vector<int> nohold(len);
    vector<int> hold(len);
    nohold[0] = 0;
    hold[0] = -prices[0];
    for (int i = 1; i < len; ++i) {
      nohold[i] = max(nohold[i-1], hold[i-1] + prices[i] - fee);
      hold[i] = max(hold[i-1], nohold[i-1] - prices[i]);
    }
    return nohold[len-1];
  }
};
